## Key features
- Fast, compiled, easy to code
- Caching & Pre-compilation
- Transpiles to JavaScript for Client-Side Rendering
- Supports embeddable code `json`, `js`, `yaml`, `css`
- Built-in Browser Sync & Reload
- Output Minifier
- Written in Nim language ğŸ‘‘
